Mancala in Java

Two student programmers and I created the Mancala board game in Java. It had almost all functionality, made use of the Model Viwer Controller pattern, and Java Swing. I programmed the parent classes, and interfaces that our program used, I helped implement the MVC pattern and the JComponents. The only non-functional feature was a special rule where if the last piece a player drop is in an empty hole on his or her side, he or she captures that piece and any pieces in the hole directly opposite of the empty hole.
